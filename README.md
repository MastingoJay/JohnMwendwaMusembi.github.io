## Purpose
This is a multipage portfolio website showcasing Data & Business Intelligence projects, technical skills, and contact information. Built with HTML5, CSS3, and JavaScript, it demonstrates responsive design, interactivity, and modern web development practices.

---

## Pages
- **index.html** — Home / Hero Section & featured projects
- **about.html** — Bio, skills, and achievements
- **portfolio.html** — Project gallery and case studies
- **contact.html** — Contact form with validation

---

## Key Features
- Fully responsive multipage layout for desktop, tablet, and mobile
- Client-side form validation (Contact page) with simulated submission
- Mobile hamburger menu with accessible aria attributes
- Scroll reveal animations using IntersectionObserver
- Active navigation highlighting
- Scroll-to-top button
- Light / Dark mode toggle with persisted preference using localStorage
- Clean and organized project structure (assets/css, assets/js, assets/images)

## Featured Projects  

1. **Boston Airbnb Market & Performance Analysis**  
   Analyzed Boston Airbnb data using Python, PostgreSQL, and Power BI to uncover host success drivers, neighborhood demand patterns, and guest experience insights. Developed an interactive dashboard for dynamic exploration, supporting data-driven growth strategies and quality improvements.  

2. **Kenya Real Estate Market Analysis**  
   Automated Excel dashboard analyzing bus transportation trends and performance. Using Power Query and PivotTables, transformed raw data into a dynamic, refreshable tool revealing key insights on bus utilization, peak hours, and year-over-year changes. Demonstrates proficiency in building data models and creating impactful dashboards for operational planning.  

3. **Wisabi ATM Transactions Report Analysis**  
   Analyzed ATM transaction data using Python, SQL, and Power BI to uncover trends in customer behavior, transaction patterns, and ATM performance. Key insights informed recommendations for ATM placement, transaction speed improvements, and digital banking adoption, enhancing customer experience and operational efficiency.  

4. **Pizza Sales Analysis — Power BI**  
   Analyzed 21,000+ pizza orders totaling $817K+ using Power BI to uncover sales trends, peak periods, and top-selling pizza categories. Developed actionable insights for pricing strategies, promotional campaigns, and inventory optimization, transforming raw sales data into data-driven business recommendations.  

5. **Movie Industry Exploratory Data Analysis**  
   Performed exploratory data analysis on a dataset of 1,000 movies using Jupyter Notebook and built an interactive Power BI dashboard to visualize trends in ratings, revenue, genres, and runtime. Insights revealed blockbuster revenue patterns, genre popularity, director/actor performance, and rating vs. revenue correlations, enabling data-driven insights into movie performance and audience engagement.  

6. **Sales Performance Analysis — Excel**  
   Analyzed global sales data in Excel to clean, structure, and visualize performance across regions, channels, and salespeople. Built an interactive dashboard using Pivot Tables, charts, and slicers to track revenue, units sold, and monthly trends. Key insights identified top-performing regions (North & South America), leading sales channels (In-store), and underperforming areas, informing strategic recommendations for growth and optimization.  

7. **India Premier League Analysis & Visualization**  
   Built an interactive IPL dashboard using PostgreSQL and Power BI to analyze player and team performance, match statistics, and season trends. Involved data cleaning, modeling, DAX calculations, and dynamic visualizations, enabling users to filter by teams, players, and seasons and explore KPIs like total runs, wickets, strike rates, and economy rates.  

8. **Nairobi Real Estate Data Scraping & Analysis — Python**  
   Scraped housing data from a Kenyan real estate website using Python and stored it in a CSV file for analysis. Extracted key property attributes such as prices and locations to identify pricing patterns across Nairobi, enabling area-based comparisons and delivering actionable market insights for buyers and investors.
